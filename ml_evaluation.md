# ML Classifier Evaluation Report

This report compares the predictions of the baseline ML Classifier (`GET /emails?classifier=ml`) against the rules-based weak labels (`GET /emails?classifier=rules`) on the Enron sample dataset.

## Summary Metrics

| Metric | Value | Description |
| :--- | :--- | :--- |
| **Total Emails** | 100 | Total number of emails evaluated. |
| **Matching Predictions** | 99 | Number of emails where ML predicted the same label as rules. |
| **Different Predictions** | 1 | Number of emails where predictions diverged. |
| **Accuracy against Weak Labels** | 99.00% | Percentage of matching classifications (ML vs Rules). |

## Evaluation Notes
- The baseline ML classifier was trained on the weak labels generated by the rules-based classifier on the Enron 100-email sample.
- An accuracy of 99% on the training set indicates that the Logistic Regression model successfully generalized and fit the keyword patterns of the weak labeler.
- Mismatches/divergences are rare on the training sample itself but provide insights into how the Logistic Regression model might prioritize vector spaces compared to rigid keyword matching.
